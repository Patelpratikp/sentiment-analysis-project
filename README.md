# Sentiment Analysis Project

## Overview
This project performs sentiment analysis on text data to classify it as positive, negative, or neutral.

## Tools & Technologies
- Python
- Pandas
- Matplotlib / Seaborn
- NLP (NLTK / TextBlob)

## Steps Performed
1. Data Cleaning
2. Text Preprocessing
3. Sentiment Analysis
4. Visualization

## Results
The model successfully classifies sentiment and provides insights from textual data.

## Conclusion
This project helped me understand NLP concepts and data analysis workflow.
